Mac使用技巧

发表于2022-08-14更新于2023-09-29字数统计1.2k阅读时长7分

调度

  • Ctrl + ↑：调度中心

    • 已修改为Command + ↑

  • Ctrl + ←或→：向左/右切换桌面

    • 已修改为Command + ←或→

  • Ctrl + ↓：当前APP的所有窗口

    • 已修改为Command + ↓

  • Control + Shift + Option + ↑：启动台

  • Control + Shift + Option + ↓：显示桌面

窗口

  • Command + Ctrl + F：切换当前APP的全屏状态

  • Command + Q：退出当前APP

  • Command + W：关闭当前APP的当前窗口

  • Command + M：最小化当前窗口到程序坞

  • Command + H：无动画隐藏当前窗口，不会占用程序坞

    • Command + H隐藏的窗口可以通过Command + Tab呼出
    • Command + M隐藏的窗口不可再次呼出，需要手动点击程序坞图标

  • Command + Option + H：隐藏除当前APP外的其他窗口

  • Command + N：打开当前APP的一个新窗口

  • 选中文件按Space：预览文件内容

    • 可以按住Space不动预览，松开Space取消预览

  • Command + Option + D：显示/隐藏程序坞

  • Command + Option + W：关闭当前APP的所有窗口

  • Command + Ctrl + Space：表情和emoji窗口

  • Command + Option + Esc：强制停止APP窗口

  • Command + ～ ：切换当前APP的窗口

  • Command + Tab：切换所有开启的APP

  • Command + Space：Spotlight聚焦搜索

  • Command + ,：打开当前APP的偏好设置

  • 按住Command键移动窗口：被移动的窗口会保持层叠层次不会跑到屏幕最前面

截图

  • Command + Shift + 3：全屏截图

  • Command + Shift + 4：区域截图

    • 已被修改为QQ同款热键：Control + Option + A
    • 区域截图后修改选区：
      • 按住Space：移动选区
      • 按住Option：修改选区

  • Command + Shift + 4 + Space：选定APP截图

  • Command + Shift + 5：完整的截图和录屏APP

访达

  • Command + Shift + G：打开前往文件夹窗口，输入路径跳转

    • 上传文件等场景时，直接把文件拖到上传窗口即可直接定位

  • Command + Backspace：删除选中文件

  • Command + T：打开新的标签页

    • 适用于多个支持多标签页的APP

  • Command + Shift + \：预览多个打开的标签页

    • 适用于多个支持多标签页的APP

  • Command + Shift + Backspace：倾倒废纸篓

  • Command + Shift + .：切换隐藏文件夹的显示

  • Option + 鼠标拖拽：拷贝文件

    • 在相同磁盘直接拖动文件是移动文件，不同磁盘拖动文件是复制文件

  • Command + 鼠标拖拽：移动文件

  • Command + Option + V：粘贴并删除源文件，相当于剪贴

  • Command + D：复制所选文件

  • Command + E：推出所选磁盘或宗卷

  • Command + F：搜索

  • Command + I：显示文件信息，相当于win的文件属性

  • Command + ：复制所选文件

  • Command + [：上一级文件夹

  • Command + ]：下一级文件夹

  • Command + ↑：定位到当前文件的文件夹

  • Command + Option + P：在底部显示/隐藏路径

    • 右键底部路径可以选择在终端中打开或复制路径

  • 按住Command拖动文件/文件夹到边栏或顶部：固定文件/文件夹到访达边栏或顶部

  • 按住Command + Shift + 点击Dock栏应用程序：在访达的应用程序中定位点击的APP

快速打开窗口

操作 作用
Shift + Command + C 打开“电脑”窗口
Shift + Command + D 打开“桌面”文件夹
Shift + Command + F 打开“最近使用”窗口
Shift + Command + G 打开“前往文件夹”窗口
Shift + Command + H 打开当前 macOS 用户帐户的个人文件夹
Shift + Command + I 打开 iCloud 云盘
Shift + Command + K 打开“网络”窗口
Option + Command + L 打开“下载”文件夹
Shift + Command + N 新建文件夹
Shift + Command + O 打开“文稿”文件夹
Shift + Command + P 在“访达”窗口中显示或隐藏预览面板
Shift + Command + R 打开“隔空投送”窗口
Shift + Command + T 显示或隐藏“访达”窗口中的标签页栏
Ctrl + Shift + Command + T 将所选的“访达”项目添加到“程序坞”
Shift + Command + U 打开“实用工具”文件夹
Option + Command + D 显示或隐藏“程序坞”
Control + Command + T 将所选项添加到边栏
Option + Command + P 隐藏或显示“访达”窗口中的路径栏
Option + Command + S 隐藏或显示“访达”窗口中的边栏
Command + / 隐藏或显示“访达”窗口中的状态栏

Safari

  • Command + T：打开新标签页
  • Command + Shift + T：恢复上次关闭的网页
  • Command + L：定位光标到地址栏
  • Command + Option + ←或→ ：切换上/下标签页
  • Command + ↑：到页面顶部
  • Command + ↓：到页面底部

终端

  • open .：用访达打开当前路径

输入

  • Control + T：交换光标前两个字符的顺序

触发角

  • 在设置 - 桌面与程序坞 - 触发角设置鼠标移动到屏幕四角时触发的操作
    • 在设置四个角的操作时可以按住修饰键防止误触

系统

  • 锁屏但不睡眠
    • Control + Command + Q：锁屏但不进入睡眠
      • 此快捷键默认被QQ的打开联系人占用
    • 直接按MacBook的锁屏按钮
  • 按住Option键点击状态栏的Wi-Fi图标：查看Wi-Fi连接的详细信息
  • 聚焦搜索结果在访达中打开：对搜索的结果按Command + Enter
  • 按住Option键点击状态栏的时间：快速开启勿扰模式
